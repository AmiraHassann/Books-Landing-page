# Yours Books Landing Page

A simple, responsive landing page for a fictional online bookstore built with only HTML and CSS (no JavaScript).

## Preview
Open `index.html` in a browser or use Live Server to see the hero, featured books grid, info highlights, and new arrivals strip.

## Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- Fonts: Playfair Display, Inter (Google Fonts CDN)
- Icons: Font Awesome 6.5.2 CDN

## Project Structure
```
index.html        # Page markup
style.css         # Styles, layout, responsive rules
img/
  photo1.avif
  photo2.avif
  photo3.avif
  photo4.avif
  photo5.avif
  photo6.avif
.vscode/
  settings.json   # Live Server port 5501
```

## Getting Started
1) Clone or download this folder.
2) Open `index.html` directly in your browser **or** run Live Server (port 5501 per `.vscode/settings.json`).

## Customization
- **Branding/text:** Edit headings, CTA labels, and logo text in `index.html`.
- **Images:** Replace the `.avif` files in `img/` or point `background-image` URLs in `index.html` to your assets.
- **Colors/radius/shadows:** Tweak CSS variables at the top of `style.css`.
- **Icons:** Update Font Awesome classes in `index.html` if you want different symbols.

## Notes
- Fully responsive using CSS Grid/Flex.
- No build step required; everything is static.
